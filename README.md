Description
-----------

WP Settings API Bootstrap is a WordPress class for plugin and theme developers to speed up development of their admin pages.

Features
--------

* Text input
* Textarea Input
* Checkbox
* Radio Button
* Multiple Checkbox
* Dropdown
* About ( just plain ol' html )
* WordPress Colorpicker
* WordPress Media Manager
